RT，按下面的操作即可：     
    
sudo apt\-get install libevent\-dev     
sudo apt\-get install python\-all\-dev     
sudo apt\-get install python\-setuptools     
sudo easy\_install greenlet     
sudo easy\_install gevent     
    
OK
